## Try-with-resources

**정의**: 자원을 자동으로 반납해주는 문법

**이점**

- **리소스 정리**: 메모리나 연결을 안전하게 해제
- **Graceful Shutdown**:
    
    **동작 순서**
    
    - 종료 신호를 받으면 새로운 요청을 거절
    - 현재 처리 중인 요청이 있으면, 해당 작업이 끝날 때까지 기다려줌
    - 요청이 끝나면 해당 빈을 파괴
    
    **Timeout**
    
    - 처리 중인 요청이 무한 루프에 빠지는 등 오래걸릴 수 있기 때문에, 보통 타임아웃을 설정함
