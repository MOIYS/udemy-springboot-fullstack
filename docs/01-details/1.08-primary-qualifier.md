## @Primary

**정의**: 여러 개의 Bean 후보가 있을 때, 우선권을 가진 Bean을 지정하는 어노테이션

**역할**

- 기본값 설정임
    - 같은 타입의 Bean이 여러 개 있어도, 스프링은 `@Primary`가 붙은 Bean 주입

**특징**

- 코드를 수정할 때, 주입받는 쪽의 코드를 건드리지 않아도 됨

<br>

## @Qualifier

**정의**: Bean에 **별명**(구분자)을 설정하는 어노테이션

**역할**

- 사용할 때, **설정한 별명**(구분자)을 불러서 주입받는 방식

**특징**

- **Bean 이름**을 **별명(구분자)**으로도 사용 가능
    - 소문자로 시작
- `@Qualifier`가 `@Primary`보다 우선순위가 높음
    - 스프링은 ‘넓은 범위의 설정보다, 좁은 범위의 상세 설정을 우선’하는 대원칙을 따름
