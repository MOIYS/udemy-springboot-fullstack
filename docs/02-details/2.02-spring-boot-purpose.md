# Spring Boot 목적

### **QUICKLY & PRODUCTION-READY**

- 빠르게 애플리케이션을 빌드하고, 프로덕션 환경에서 사용 가능한 애플리케이션을 원함

<br>

### 1) Build QUICKLY

**1. Spring Initializr**

**역할**: 스프링 부트 프로젝트의 뼈대를 쉽게 만들어주는 웹 기반 도구

<br>

**2. Spring Boot Starter Projects**

**역할**: 특정 기능 구현에 필요한 모든 라이브러리(의존성)와 자동 설정을 하나의 패키지로 묶어 제공하는 모듈

<br>

**3. Spring Boot Auto Configuration**

**역할**: Bean을 자동으로 등록해주는 기능

<br>

**4. Spring Boot DevTools**

**역할**: 개발 생산성을 향상시키는 모듈. 서버 자동 재시작, LiveReload 기능 등을 제공

<br>

### 2) Be PRODUCTION-READY

**1. Logging**

**역할**: 로그 시스템 제공

<br>

**2. Different Configuration for Different Environments**

**역할**: 개발(dev), 운영(prod), 테스트(test) 환경에 따라 다른 설정 적용 기능

- **Profiles**: 환경에 따라 다른 설정을 적용할 수 있게 하는 기능   
- **ConfigurationProperties**: 설정 파일에 정의된 설정값들을 자바 클래스로 바인딩 해줌    

<br>

**3. Monitoring (Spring Boot Actuator)**

**역할**: 애플리케이션 내부 정보를 모니터링 할 수 있는 기능
