# Spring Boot Starter Projects

**정의**: 특정 기능 구현에 필요한 모든 라이브러리(의존성)와 자동 설정을 하나의 패키지로 묶어 제공하는 모듈

<br>

### **필요성**

- REST API 빌드, Unit Test들을 위한 많은 프레임워크들이 필요함
    - → 해당 프레임워크들을 그룹화해서 애플리케이션을 쉽게 빌드할 수 있게 해줌

<br>

### **특징**

**1. Spring Boot는 다양한 Starter Project를 제공**

- `spring-boot-starter-web`, `spring-boot-starter-data-jpa`, `spring-boot-starter-test` 등

<br>

**2. 버전 관리 자동화**

- 라이브러리들의 버전 호환성을 자동으로 맞춰줌

<br>

**3. 전이적 의존성(Transitive Dependencies)**
- `Starter` 하나를 추가하면, 그 Starter가 피요로 하는 다른 라이브러리들까지 자동 다운로드
- e.g. `spring-boot-starter-web`을 추가하면
`spring boot starter`, `spring-boot-starter-json`, `spring-boot-starter-tomcat`, `spring-web`, `spring-webmvc`가 있음

<br>

4. **Spring Boot Starter Project는 편리한 의존성 디스크립터**
