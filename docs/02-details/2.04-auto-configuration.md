# Auto Configuration

**정의**: Bean을 자동으로 등록해주는 기능

- 애플리케이션용 자동화 설정
- 클래스 경로에 있는 프레임워크에 따라 생성

<br>

### CONDITIONS EVALUATION REPORT

**정의**: 스프링 부트 애플리케이션 시작 시, 어떤 항목이 **자동 설정되었고(Positive)**, 어떤 것은 **설정되지 않았는지(Negative)** 알려주는 로그 리포트

- **Postive matches**: 자동 **설정된** 항목
- **Negative matches**: 자동 **설정되지 않은** 항목

<br>

### `spring-boot-starter-web` 자동 설정 예시

- Dispatcher Servlet (`DispatcherServletAutoConfiguration`)
- Servlet Container (`EmbeddedWebServerFactoryCustomizerAutoConfiguration`)
- Default Error Pages (`ErrorMvcAutoConfiguration`)
- Json 변환 (`JacksonHttpMessageConvertersConfiguration`)
- MVC 관련 설정 (`WebMvcAutoConfiguration`) 등
