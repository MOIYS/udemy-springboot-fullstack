# Spring Framework / Spring MVC / Spring Boot

### 1) Spring Framework

**정의**: Java 애플리케이션 개발을 위한 프레임워크

<br>

**핵심 기능**

- **DI & IoC**
- **트랜잭션 관리** 등

<br>

### 2) Spring MVC (Spring Module)

**정의**: 웹 애플리케이션 개발을 위한 **MVC 디자인 패턴** 기반 프레임워크

- **Spring Framework 안에 포함**된 하나의 **모듈**

<br>

**핵심 기능**

- **HTTP 요청 처리**
- **DispatcherServlet**
- **@Controller** 등

<br>

### 3) Spring Boot (Spring Project)

**정의**: 복잡한 Spring 설정을 자동화하여, 실행 가능한 애플리케이션을 빠르고 쉽게 만들 수 있도록 도와주는 프레임워크

<br>

**핵심 기능**

- **자동 설정 (Auto Configuration)**
- **내장 서버 (Embedded Server)**
- **의존성 관리 (Starter)**

<br>
<br>

# Embedded Server


**정의**: **애플리케이션 내부에 포함**된 웹 서버. 별도로 설치하지 않아도 됨

<br>

### 배포 방식

**1. WAR 방식** 
- Java 설치 → 서버(WAS) 별도 설치 → WAR 파일로 빌드 → WAS의 배포 디렉토리에 WAR 복사 → WAS 실행

<br>

**2. Embedded Server 방식**
- Java 설치 → JAR 파일로 빌드 → 서버에 JAR 파일 복사 → `java -jar {my-app}.jar` 명령어 또는 Docker로 실행
